Overview

    The goal of the programming exercise is to build a simple web application that interacts with an API to:
  • View all the records stored in a database table
  • View a single record
  • Create new records
  • Edit entries
  • Delete entries

Basic Activities

  • Create a mock JSON API
  • Develop list page for all table entries
  • Develop review page for table
  • Develop create page for table
  • Develop edit page for table
  • Develop delete page for table
